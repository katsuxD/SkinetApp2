Internal Server Error - Làm mới lại trang sẽ khiến lỗi biến mất

Error: {{error.message}}

Lưu ý: Nếu như bạn đang thấy dòng thông báo này tức là đã có lỗi bên phía Angular

Cần làm gì tiếp theo?

  1. Hãy bật chrome dev tools
  2. Tiếp theo nhấn vào network tab
  3. Kiểm tra những request bị lỗi trong mục failing request
  4. Kiểm tra đường dẫn URL đó - Hãy chắc rằng bạn đang nhập chính xác
  5. Tái hiện lại lỗi trong postman - Nếu trong Postman có lỗi tương tự, vậy thì lỗi không phải đến từ phía Angular =]] !

Cứ thực hiện theo từng bước như trên - Đấy là lúc bạn đã nhận ra lỗi đến từ đâu và xử lý nó như thế nào ^^
